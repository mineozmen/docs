# Exercise: To-do List

Examples provided within this section provide details of building a new App, with:

* [**To-do List Runner**](to-do-list-runner.md)**:** Development and deployment of a new microservice for CRUD operations on a todo MongoDB collection
* [**To-do List Gateway**](to-do-list-gateway.md)**:** Configuration of an existing gateway to map external requests to the new CRUD microservice
* [**To-do List UI**](to-do-list-ui.md)**:** Definition of source, data structure and UI components to allow interaction with the to-do list from admin UI
* [**To-do List Query**](to-do-list-query.md)**:** Extension of the basic To-do List features to allow search and display of todo list results
