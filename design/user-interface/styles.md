# Styles

<figure><img src="../../.gitbook/assets/image (147).png" alt=""><figcaption><p>Style UI</p></figcaption></figure>

Styles can be defined with the following configurations:

* **Element:** Any valid CSS element reference (e.g. using tag, class, id)
* **Key-Value:** CSS attributes (including variable definitions) to apply for the element
